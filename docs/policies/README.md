# TurqApp Uygulama Ici Politika Paketi

Bu klasor, uygulama icinde gomulu olarak gosterilebilecek Turkce politika metinlerini icerir.

Bu set bilerek:

- asiri kurumsal dile kacmaz
- adres, MERSIS, vergi numarasi gibi agir sirket alanlarina dayanmaz
- uygulama ici okuma deneyimine uygun, kisa baslikli ve acik bir yapida tutulur
- TurqApp'in sosyal, egitim, burs, is ilanlari, mesajlasma ve pazar yeri yuzeylerini birlikte kapsar

Dosyalar:

- `AYDINLATMA_METNI_TR.md`
- `GIZLILIK_POLITIKASI_TR.md`
- `KULLANIM_KOSULLARI_TR.md`
- `TOPLULUK_KURALLARI_TR.md`
- `ICERIK_GUVENLIGI_VE_MODERASYON_POLITIKASI_TR.md`

## Uygulama Ici Gosterim Notu

Bu metinler web sayfasi diliyle degil, uygulama ici politika ekranlari icin yazildi.

Onerilen ekran sirasi:

- Politikalar
- Gizlilik Politikasi
- Aydinlatma Metni
- Kullanim Kosullari
- Topluluk Kurallari
- Icerik Guvenligi ve Moderasyon

## Hala Netlestirilmesi Gereken Basliklar

Politika metinleri genel ve kapsayici tutuldu. Yayin oncesi urun seviyesinde ayrica netlesmesi gereken konular:

- 18 yas alti kullanicilar icin urun kurali
- yurt disi veri aktarimi olup olmadigi
- hesap silme ve veri talep akisinin uygulama ici yeri
- destek ve raporlama icin kullanilacak tek iletisim kanali
