## Patchable Scope

The current audit splits patch scope into two buckets:

- total patchable posts: `31,062`
- phase-1 safe posts: `26,632`
- phase-2 image posts: `4,430`
- patchable `thumbnail` fields: `26,632`
- patchable `img[]` fields: `4,430`
- patchable `video` fields: `0` in phase-1 URL backfill

This patchable scope is based only on rows where the legacy tokenized URL maps
cleanly back to `Posts/{docId}/...`.

## Decisions

1. Keep short-link prefixes as-is.
2. Treat `https://cdn.turqapp.com/Posts/...` as the canonical public post asset
   contract.
3. Stop functions from emitting tokenized post asset URLs.
4. Backfill `thumbnail` first.
5. Treat the `2` failed-HLS MP4 rows as a separate repair queue.
6. Complete `img[]` canonicalization after public-read rollout validation.
7. Canonicalize user avatars under `users/...`, then propagate that reference
   into post `authorAvatarUrl` / `avatarUrl`.
